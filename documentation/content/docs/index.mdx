## Overview

Weaver wraps a live Stitch WebView session with a native Compose shell: canvas nodes from the editor, offline cache, export to the bridge, and fold-aware layouts.

See the [landing page](/weaver) for foldable feature walkthroughs (Tabletop Studio, Fold-to-Compare, Fire-and-Fold).
